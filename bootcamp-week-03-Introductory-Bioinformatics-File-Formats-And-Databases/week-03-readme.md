# 🧬 Week 3: Introductory Bioinformatics – File Formats & Databases

Welcome to **Week 3** of the Computational Biology Bootcamp!  
This week, we explore the fundamental building blocks of bioinformatics: **file formats**, **databases**, and **terminology** that every bioinformatician must know.

---

## 📚 Overview

This module introduces:

### 📄 Key Bioinformatics File Formats
- **FASTA** – Standard for nucleotide or protein sequences
- **VCF (Variant Call Format)** – Describes genetic variants
- **BAM (Binary Alignment/Map)** – Stores aligned sequencing data efficiently

### 🗂️ Major Bioinformatics Databases
- **TCGA (The Cancer Genome Atlas)** – Large-scale cancer genomic dataset
- **Ensembl Genome Browser** – Genomic data for vertebrates and other eukaryotic species
- **UCSC Genome Browser** – Visual and searchable access to genome sequences and annotations

### 📦 Why Standardization Matters
- Facilitates **reproducibility**
- Ensures **data integrity** and **collaboration**
- Makes integration across pipelines and tools easier

---

## 💡 Learning Objectives

By the end of this week, you should be able to:

- Define key bioinformatics terms and concepts  
- Understand the purpose and structure of commonly used file formats  
- Explore public data repositories like TCGA, Ensembl, and UCSC  
- Recognize the importance of standard data formats and databases for reproducible research
